# Knowledge

Your personal knowledge base, organized by the PARA framework.

See `framework/principles/para-knowledge-organization.md` for
the filing rules.

## Structure

- `projects/` — Active projects (work, personal, creative)
- `areas/` — Ongoing responsibilities (no end date)
- `resources/` — Topics of interest, reference material
- `archive/` — Inactive items from the above three
- `_context/` — Save-memory snapshots (system utility)
